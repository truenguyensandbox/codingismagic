🖥️ Bash – Command Line Conjuring

The terminal is your altar. Bash is your spell language. Here, you command the system itself.

🔮 What is Bash?

Bash is a command-line shell used in Unix and Linux systems. It’s where real wizards go to cast spells directly — file manipulation, automation, navigation, system control. All in pure rune form.

🗂️ Move, Summon, Delete

cd spells/
ls
rm cursed-scroll.txt
mv draft.txt scroll.md

Bash commands are fast, powerful, and dangerous — use them with intention.

📜 Scripting Spells

Bash scripts are files full of commands you can run all at once — like a ritual sequence.

#!/bin/bash
echo "🪄 Welcome, Mage!"
mkdir archive
mv *.log archive/
echo "Logs sealed."

⚡ Chaining and Logic

You can combine spells using && (if success), pipes | (pass output), or control structures like loops.

ls | grep "spell"
mkdir backup && echo "Backup created"
