⚡ JavaScript – The Incantation of Action

This is the spark. The whisper that makes your spell respond, change, and think.

✨ What is JavaScript?

JavaScript is a language of logic and interaction. With it, you animate your glyphs, bind spells to buttons, and make your pages breathe.

HTML builds the bones. CSS adds beauty. JavaScript gives it a soul.

🌀 Functions – Reusable Action Scrolls

A function is a named spell. Write it once, cast it whenever you need it.

function castSpell(name) {
  console.log("✨ You cast " + name + "!");
}

castSpell("Lumos");
castSpell("Alohomora");

🎯 Events – Reactive Magic

Your code can listen for clicks, keys, scrolls — and react. This is how your spell feels alive.

document.querySelector(".wand").addEventListener("click", () => {
  alert("✨ Wand activated!");
});

🌿 The DOM – The Living Tree of the Page

JavaScript doesn’t just act. It speaks to the DOM — the Document Object Model — a live version of your HTML page that JS can touch, change, and grow.

You’re not just coding. You’re conversing with a living structure.
