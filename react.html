🧩 React – Reusable Interface Magic

React lets you build interactive enchantments from reusable UI spell scrolls.

✨ What is React?

React is a JavaScript library that lets you build components — pieces of your interface that update themselves when the state changes. It’s like creating enchanted UI scrolls that re-render with every breeze of data.

📦 Components – Custom Spell Scrolls

A React component is a reusable bit of interface magic. It can hold its own logic, structure, and style — and be summoned wherever you need it.

function SpellButton() {
  return <button>✨ Cast Spell</button>;
}

🧪 Props – Spell Ingredients

Props are how you pass unique ingredients into your spell — allowing each scroll to behave slightly differently.

function Spell({ name }) {
  return <p>You cast {name}!</p>;
}

⚙️ State – Reactive Mood Shifts

A component can change over time. With useState(), your UI responds to user input or internal changes like a spell that re-casts itself.

const [mana, setMana] = useState(100);

<button onClick={() => setMana(mana - 10)}>
  🔥 Cast – Mana Left: {mana}
</button>
