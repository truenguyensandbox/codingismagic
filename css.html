🎨 CSS – Glamour Charms

With CSS, your structure begins to shimmer. Form becomes function, and function becomes beautiful.

✨ What is CSS?

CSS stands for Cascading Style Sheets, but here, we call it the glamour charm of web magic. It transforms raw structure into shimmering design — adjusting color, layout, spacing, motion, and more.

🎯 Selectors = Targeting Spells

To cast a style, you must target the element. Use its tag, class, or ID to focus your charm.

p {
  color: #F4ACB7;
}

This targets all paragraph tags with a soft blush of magical text color.

🧪 Properties = Style Ingredients

Each declaration includes a property and a value — like choosing your spell ingredients and amounts.

.spell-card {
  background: #FFF0F3;
  border-radius: 1rem;
  box-shadow: 0 4px 10px rgba(0, 0, 0, 0.1);
}

This wraps your element in a soft, glowing charm.

🌊 The Cascade

If multiple spells apply to the same element, the cascade determines which one wins. More specific selectors win over general ones. Later rules override earlier ones.
