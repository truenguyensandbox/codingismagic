⏳ Git – The Time-Bending Spellbook

Git lets you save your work like pages in a magical tome — go back, try again, or split into alternate timelines.

📖 What is Git?

Git is a version control system — it tracks changes to your code over time. Every saved change is a commit, like a bookmarked page in your spellbook. You can go back, branch off, and merge changes like a time mage.

💾 Commits – Save Your Progress

Each time you commit, you save your spell's current state. It's like writing in your grimoire.

git add .
git commit -m "✨ Added fireball spell logic"

🌿 Branches – Try Alternate Timelines

Branches let you explore new ideas without breaking the original timeline. If your experiment works, you can merge it back in. If it fails, no worries — you’ve preserved your history.

git checkout -b lightning-spell

🧵 Merge – Reweave Timelines

Once your new spell is ready, you can merge it into your main book:

git checkout main
git merge lightning-spell

The timelines combine, and your spellbook evolves.

🌍 GitHub – Share Your Grimoire

GitHub is a remote library where you can store your spellbook online, collaborate with others, and showcase your work to the world.

git remote add origin https://github.com/yourname/yourspellbook.git
git push -u origin main
