📜 HTML – Structure Glyphs

HTML is the rune language that builds the skeleton of your spell.

✨ What is HTML?

HTML stands for HyperText Markup Language — but here, we call it the structure spell. It creates order, shapes content, and tells the browser what each piece means.

🔤 Tags = Summoning Runes

HTML uses tags — glyphs inside angle brackets — to summon elements into existence.

<h1>Summon a heading</h1>
<p>Create a paragraph of magic.</p>
<a href="portal.html">Open a portal</a>

Each tag opens a gate. Some have attributes (like href) to guide their power.

🧱 Elements = Enchanted Blocks

When a tag is activated, it forms an element — a real part of your page.

<img src="crystal.jpg" alt="Glowing Crystal" />

This line conjures an image. The alt attribute provides fallback magic.

🔁 Nesting = Containment Spells

You can place glyphs within glyphs — paragraphs in sections, list items in lists.

<ul>
  <li>Potion ingredients</li>
  <li>Wand polish</li>
</ul>

This nesting creates structure and hierarchy. Like placing scrolls in shelves in rooms.
